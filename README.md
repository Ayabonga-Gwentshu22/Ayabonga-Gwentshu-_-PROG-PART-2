The QuickChat application is a Java messaging program created using object-oriented programming principles. It allows users to register, log in, and send messages through a console-based system in NetBeans. The program is made up of three main classes: LoginForm, Message, and MessageTest, each handling a different part of the application. The LoginForm class manages user registration and validates usernames, passwords, and South African cellphone numbers. The Message class handles creating messages, checking message length and recipient numbers, generating message IDs and hashes, and storing messages in a JSON file. The MessageTest class controls the main flow of the program, including menu options and user interaction.Overall, the application shows the use of important Java concepts such as classes, methods, loops, validation, file handling,and user interaction to create a working messaging system.
